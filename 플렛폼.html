리모델링 투자 분석 플랫폼

건축물대장, 지도, 공공데이터, 실거래가, AI 분석을 결합하여 건물의 리모델링 사업성을 자동으로 분석하는 투자 의사결정 플랫폼

1. 서비스 개요

사용자가 주소를 검색하거나 지도에서 건물을 선택하면 해당 건물의 건축물대장, 토지정보, 편의시설, 실거래가를 수집하고 AI가 리모델링 비용과 미래 가치를 분석하여 투자 수익성을 예측하는 시스템

2. 메인 화면 구성

지도 기반 분석

Google Map 또는 Kakao Map 📍 건물 위치 표시 📍 투자등급 표시 📍 ROI 표시 📍 공사비 예상 표시

우측 상세 패널

항목 내용
건축물 정보 건축년도, 대지면적, 연면적, 용적률, 건폐율
편의시설 분석 지하철, 학교, 마트, 병원, 공원 거리
사업성 분석 현재가치, 공사비, 미래가치, ROI
투자등급 A~D 등급 자동 산정

3. 투자 분석 로직

분석항목 설명
현재 가치 실거래가 및 주변 시세 기반
공사비 AI 예측 또는 실측 견적
미래 가치 리모델링 완료 후 예상 가치
차익 미래가치 - 현재가치 - 공사비
ROI 예상 투자수익률

4. 공사비 산정 방식

1차 분석 (AI 예측)

  • 건축년도
  • 건물용도
  • 연면적
  • 지역
  • 건물등급
  • 유사 건물 공사이력
AI가 방문 없이 추정하는 비용 예시) 외벽공사 : 2.1억 옥상방수 : 0.8억 공용부 : 3.2억 내부공사 : 5.4억 기타 : 2.5억 총 공사비 : 14억

2차 분석 (실측 기반)

  • 현장 방문
  • 사진 촬영
  • 실측 데이터 입력
  • 노후도 평가
  • 배관 상태
  • 전기 상태
  • 승강기 상태
  • 외벽 상태
실측이 완료되면 예측치 → 확정 견적 으로 변경

9. 장기 비전

초기 건축물대장 + 지도 + 공사비 분석 ↓ 중기 리모델링 투자 분석 플랫폼 ↓ 후기 건물 매입 의사결정 AI ↓ 최종 건물 투자·리모델링·매각 전 과정을 지원하는 AI 기반 PropTech 플랫폼
